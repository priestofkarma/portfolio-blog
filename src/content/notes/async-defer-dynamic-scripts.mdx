# Async, defer, динамические скрипты

Скрипты с аттрибутом defer никогда не блокируют страницу, скрипты с этим атрибутом загружаются в фоновом режиме, и запускаются только после загрузки всего html.

Async скрипт загружается в фоновом режиме, и после загрузки, сразу же запускаются. Такой атрибут лучше ставить для скриптов, которые не зависят от других скриптов, и не работают с элементами на странице.

import * as React from "react";
import { graphql } from "gatsby";

const IndexPage = ({ data }) => {
  return (
    
    <main>
      <p>This site was last built on:</p>
      <p>{data.site.buildTime}</p>
    </main>
  );
};

export default IndexPage;

export const query = graphql`
  query {
    site {
      buildTime(formatString: "DD/MM/YYYY")
    }
  }
`;
