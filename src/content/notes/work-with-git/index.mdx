### Ініціалізація та пуш на сервер

```bash
// Ініціалізувати репозиторій
git init

// Локальний конфіг для репозиторію
// Для глобального конфігу прописати прапор --global
git config --local user.name "Zhenya"
git config --local user.email "email@gmail.com"

// Перевірка статусу
git status

// Додати ВСІ файли в індекс
git add -A // додає всі файли до індексу

// Зробити коміт
git commit -a -m"first commit" // (-a) додає коміт для всіх файлів

// Перевірити логі проекту
git log

// Запустити зміни
git push

// Зв'язати локальний репозиторій із гітом на сервері
git remote add origin "https://github.com/preistofkarma/reponame.git"
// Вибрати гілку main (перевіменувати гілку master на main)
git branch -M main
// Запустити зміни 
git push -u origin main // прапор -u означає, що всі наступні пуші будуть відправлятися в main
```

### Клонування репозиторію

```bash
// Клонування репозиторію в діректорію reponame
git clone https://github.com/preistofkarma/reponame.git reponame

// Отримати зміни віддаленого сервера
git pull

// Змінити посилання на репозиторій для пуша
git remote set-url origin https://github.com/preistofkarma/reponame.git
```
